## ğŸ“– About This Repository

The **pyfundamentals-journey** repository is a quick-reference guide for anyone looking to learn or review Python concepts and functions efficiently.

This repository is inspired by the book **[Python Crash Course](https://www.amazon.com/Python-Crash-Course-3rd-Eric-Matthes/dp/1718502702)**. Each folder corresponds to a chapter in the book, with the exception of the first, which is dedicated to setting up your Python and Jupyter development environment.

## ğŸš€ Getting Started

All exercises were developed using Jupyter Notebooks to facilitate organization and structure. To run any `.ipynb` file, you need to have Jupyter installed.

1.  **Install Jupyter:**
    ```bash
    pip install jupyter
    ```

2.  **Run a Notebook:**
    Navigate to the folder containing the notebook and run:
    ```bash
    jupyter notebook <filename>.ipynb
    ```
    Replace `<filename>.ipynb` with the name of the notebook you want to open.

The projects proposed in the book will be created in separate repositories for better organization and version control.
